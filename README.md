# Home
Essa é uma aplicação que é voltada para o estudo de Micro-frontends, usando Module Federation.

_Esse micro-frontend é responsável pela Home_

Essa aplicação é executada na porta ```10002```.

## Instalando
```
npm i
```

## Executar o código
```
 npm run dev
```
